Leveraging AI and PostgreSQL for an Inclusive Neurodivergent Career Platform

← Back to Projects

NeuroBridge is a web-based career support and skill assessment platform designed to assist neurodivergent individuals in identifying suitable career paths through structured assessments and data-driven analysis.

Objective

The goal of NeuroBridge is to bridge the gap between neurodivergent students and employment opportunities by providing structured skill evaluation, accessibility support, and data-driven career guidance.

Research Foundation

The project is supported by research studies on assistive technology, workplace accessibility, and employment challenges faced by autistic and neurodivergent individuals.

System Architecture

  • Frontend: React
  • Backend: Supabase
  • Database: PostgreSQL

Core Features

  • Domain-specific Skill Assessment Modules
  • Structured Data Collection and Result Storage
  • Accessible UI Design
  • Manager Login for Monitoring
  • Integration with Job Platforms

Implementation Approach

The system collects user responses and stores them in PostgreSQL. Backend logic processes responses for structured evaluation.

Outcome

NeuroBridge demonstrates inclusive technology integration to improve employment accessibility.
