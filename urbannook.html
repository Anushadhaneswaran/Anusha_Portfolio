Project Overview

UrbanNook Clothing is a full-stack e-commerce web application developed during my internship at CodeAlpha. The project demonstrates complete integration of frontend, backend logic, and relational database management.

Objective

To design and implement a dynamic e-commerce system capable of handling user authentication, product management, cart functionality, and order processing.

System Architecture

  • Frontend: HTML, CSS, JavaScript
  • Backend: Express.js (Node.js)
  • Database: MySQL

Core Functionalities

  • User Registration and Secure Login
  • Profile Management
  • Dynamic Product Listing
  • Shopping Cart System
  • Order Processing

Outcome

Successfully demonstrated full-stack development including REST APIs, database connectivity, and scalable architecture.
